# Prince Food Paradise

Welcome to **Prince Food Paradise**, a web-based application designed to manage food items, allowing users to add, update, and delete menu items while also enabling customers to order from various food sections.

## Features

- **Add Food Items**: Admin can add new food items with details such as name, price, section, and image URL.
- **Delete Food Items**: Admin can delete existing food items by name.
- **Update Food Items**: Admin can update the details of existing food items.
- **Categorized Menu Sections**: Food items are organized into sections: Tiffin, Meals, and Fast Food.
- **Add to Cart**: Users can add items to their cart for ordering.
- **Total Calculation**: Displays total price of items in the cart.
- **Pay and Print Bill**: Users can pay their bill and print it.

## Technologies Used

- **HTML**: For structuring the web page.
- **CSS**: For styling the application.
- **JavaScript**: For interactivity and functionality.
- **Tailwind CSS**: For responsive design and utility-first CSS framework.
- **Font Awesome**: For icons and user-friendly interface.

Usage
Adding Items:

Fill out the "Add product/item" section with the item name, price, section, and image URL.
Click the Add button to add the item to the specified section.
Deleting Items:

Enter the item name in the "Delete Item" section and click the Delete button.
Updating Items:

Fill in the old item name and the new details in the "Update Item" section and click the Update button.
Adding to Cart:

Click on the Add to cart button next to any item to add it to your cart.
Paying and Printing Bill:

Click on Pay Bill to finalize your order or Print Bill to print the current bill.
